# 🤖 Robo Clicker

**Космический браузерный кликер о развитии лаборатории будущего**

Кликай по роботу, зарабатывай золото, усиливай производство, открывай скины, используй бусты и выполняй серии достижений.

## Об игре

**Robo Clicker** — браузерная incremental/clicker-игра. Игрок становится главным инженером секретной лаборатории будущего и развивает её от первых ручных кликов до автоматизированной роботизированной системы.

Основной игровой цикл:

1. Нажимать на робота и получать золото.
2. Улучшать силу клика.
3. Покупать роботов для автоматического дохода.
4. Повышать уровень инженера.
5. Открывать скины, бусты и достижения.
6. Возвращаться в игру и получать накопленный офлайн-доход.

## Игровой процесс

### Ручной доход

Нажатие на центрального робота начисляет золото. Количество монет зависит от:

- базовой силы клика;
- купленных улучшений;
- уровня игрока;
- постоянных усилений;
- временных бустов;
- критических эффектов.

### Улучшение клика

Кнопка **«Улучшить клик»** повышает базовую силу нажатия. Цена следующего улучшения увеличивается по мере развития.

Начальная стоимость улучшения:

```text
85 монет
```

### Роботы

Роботы создают автоматический доход каждую секунду.

Начальная стоимость первого робота:

```text
200 монет
```

Количество роботов, их цена и доход сохраняются между сессиями.

### Уровни

Первый переход с `LVL 0` на `LVL 1` требует:

```text
150 кликов
```

Требование следующего уровня растёт с множителем:

```text
2.1
```

Прогресс отображается на отдельной полосе уровня.

## Офлайн-доход

После возвращения в игру рассчитывается доход, созданный роботами во время отсутствия игрока.

Максимальный учитываемый период:

```text
2 часа
```

Буст **«Офлайн-ускоритель»** может увеличить доход за часть этого периода.

## Скины

В игре доступно 16 скинов разной редкости:

- common;
- uncommon;
- rare;
- ultra.

Первый классический робот доступен бесплатно. Остальные скины покупаются за игровую валюту, открываются наградами или через rewarded-рекламу.

Выбранный скин сохраняется между запусками.

## Бусты

Бусты разделены на три категории:

| Категория | Назначение |
|---|---|
| Временные | Усиливают клики, роботов или офлайн-доход на ограниченное время |
| Постоянные | Навсегда улучшают основные характеристики |
| Супер | Дают мощные кратковременные эффекты и комбинации |

В проекте реализованы:

- **Неоновый разгон** — множитель силы клика;
- **Офлайн-ускоритель** — усиление офлайн-дохода;
- **Армия дронов** — усиление дохода роботов;
- **Золотой шторм** — дополнительный доход за клик;
- **Улучшенный процессор**;
- **Вечный генератор**;
- **Модуль эволюции**;
- **Космический усилитель**;
- **Критический перегруз**;
- **Заморозка времени**;
- **Галактический прорыв**;
- **Омега-режим**.

Таймеры активных бустов приостанавливаются во время рекламы и внешней паузы игры.

## Достижения

В игре предусмотрено 80 этапов достижений, сгруппированных в тематические серии:

- клики;
- заработанные монеты;
- роботы;
- улучшения;
- сила клика;
- уровни;

- легендарные цели.

За завершение этапов игрок получает монеты и дополнительные награды, включая постоянные бонусы и скины.

## Управление

### Компьютер

| Действие | Управление |
|---|---|
| Получить золото | Клик по роботу |
| Открыть раздел | Клик по кнопке |
| Изменить настройку | Мышь или клавиатура |
| Прокрутить окно | Колесо мыши |

### Мобильное устройство

| Действие | Управление |
|---|---|
| Получить золото | Касание робота |
| Навигация | Касание кнопок |
| Прокрутка модального окна | Вертикальный свайп |

### Smart TV

При обнаружении телевизионного браузера доступны:

- стрелки — переход между элементами;
- `Enter` — активация;
- `Escape` или `Backspace` — закрытие окна или возврат назад.

## Звук

Проект использует три аудиофайла:

| Файл | Событие |
|---|---|
| `click.wav` | Нажатие на робота |
| `start.wav` | Начало игры |
| `menu.wav` | Возврат в меню |

Общая громкость регулируется в настройках и сохраняется в браузере.

## Локализация

Поддерживаемые языки:

- русский;
- английский.

При запуске язык выбирается по окружению Yandex Games или языку браузера. После этого игрок может изменить язык вручную в настройках.

## Сохранение прогресса

Основное состояние хранится в `localStorage`.

Сохраняются:

- монеты;
- сила клика;
- цены улучшений;
- роботы и автоматический доход;
- время последней сессии;
- уровень и прогресс;
- громкость, яркость и тема;
- открытые и выбранный скин;
- бусты и их таймеры;
- общая статистика;
- достижения и полученные награды.

При авторизации в Yandex Games выполняется частичная облачная синхронизация основных показателей игрока.

> Очистка данных сайта в браузере удалит локальное сохранение.

## Yandex Games SDK

В проекте используются:

- `LoadingAPI.ready()`;
- `GameplayAPI.start()` и `GameplayAPI.stop()`;
- получение объекта игрока;
- частичная запись данных игрока;
- баннерная реклама;
- полноэкранная реклама;
- rewarded-реклама;
- события внешней паузы и возобновления игры.

При отсутствии SDK игра запускается в локальном режиме, но рекламные функции и облачная синхронизация недоступны.

## Локальный запуск

Из-за подключения `/sdk.js` проект лучше открывать через локальный HTTP-сервер, а не напрямую через `file://`.

### Python

Откройте терминал в корне проекта:

```bash
python -m http.server 8000
```

На некоторых системах:

```bash
python3 -m http.server 8000
```

Затем откройте:

```text
http://localhost:8000
```

### Visual Studio Code

Можно использовать расширение **Live Server**:

1. Открыть папку проекта.
2. Нажать правой кнопкой по `index.html`.
3. Выбрать **Open with Live Server**.

При локальном запуске игра работает без Yandex SDK. Настоящую рекламу и облачную синхронизацию необходимо проверять в окружении Yandex Games.

## Публикация

Для публикации в Yandex Games подготовьте архив, в корне которого находятся:

```text

```

Не помещайте файлы внутрь дополнительной вложенной папки.

Перед публикацией рекомендуется проверить:

- запуск из релизного архива;
- мобильную и альбомную ориентацию;

- офлайн-доход;
- все рекламные callbacks;
- паузу звука и таймеров во время рекламы;
- русскую и английскую локализацию;
- сброс прогресса;
- долгую игровую сессию.
