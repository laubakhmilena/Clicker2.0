# Тест-кейсы Robo Clicker

## TC-001 — Загрузка приложения в локальном режиме

| Поле | Значение |
|---|---|
| Тип | Smoke |
| Приоритет | P0 |

**Предусловие**  
Открыть корень проекта через HTTP-сервер.

**Шаги**  
1. Перейти на URL приложения.  
2. Дождаться DOMContentLoaded.  
3. Проверить главное меню и Console.

**Ожидаемый результат**  
Отображается Robo Clicker; нет необработанных ошибок; отсутствие SDK не блокирует игру.
