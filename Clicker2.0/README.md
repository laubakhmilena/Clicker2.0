# Clicker 2.0

Clicker 2.0 — обновленная версия браузерной clicker-игры. Игрок нажимает на объект, зарабатывает очки, улучшает прогресс и постепенно открывает новые возможности.

## Что внутри

- игровой экран на HTML/CSS/JavaScript;
- звуковые эффекты для кликов, меню и старта;
- файл `roadmap.md` с планом развития;
- `deploy.bat` для подготовки/запуска вспомогательных действий;
- архив `Clicker2.0.zip` с версией проекта.

## Структура

```text
index.html       разметка интерфейса
style.css        стили игры
script.js        логика кликера
roadmap.md       план доработок
*.wav            звуки
deploy.bat       вспомогательный скрипт
```

## Запуск

Откройте `index.html` в браузере. Если звуки или сохранения работают нестабильно при открытии файлом, запустите проект через локальный сервер.
